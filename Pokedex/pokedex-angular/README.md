# Pokédex Angular (Standalone)

Proyecto basado en el tutorial. Incluye:
- Lista con buscador (1ª gen. 1-151)
- Detalle con navegación anterior/siguiente
- Componente de tipo con colores
- Pipe `digitNumber`

## Requisitos
- Node.js LTS 18 o 20
- Angular CLI 18 (`npm i -g @angular/cli`)

## Cómo ejecutar
```bash
npm install
npm start
# luego visita http://localhost:4200
```
